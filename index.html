This is a boilerplate for use in Github Pages. It is based on The HTML5 Boilerplate, and comes with jQuery and text.css from 960.gs. Octocat icon from designmoo.

Instructions

  1. fork this repo
  2. rename this repo to user.github.com
  3. edit _config.yml
  4. edit index.html
  5. edit _includes/style.css
  6. navigate to user.github.com and enjoy
